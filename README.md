# Nominal Sets #

We aim to have a library of nominal sets following Andrew Pitts'
"Nominal Sets". As far as possible we use the [standard
library](https://github.com/agda/agda-stdlib/).

## Tasks ##

  * Permutations
  * Actions and G-set
  * Category of G-sets
